## Interpreting Results

| Result | Meaning |
|--------|---------|
| ✅ PASS | Component works as expected |
| ⚠️ WARN | Works but with caveats (e.g., low confidence) |
| ❌ FAIL | Component broken or misconfigured |

**Common failures:**
- `ChromaDB not found` → Run `app/build_memory_index.py`
- `Model not found` → Run `ollama pull <model>`
- `GEMINI_API_KEY not set` → Add key to `.env`
- `Import error` → Run from `project-recall/` directory
